> A comprehensive, curated list of brutalism resources spanning architecture, web design, graphic design, games, media, and culture

Brutalism is both an architectural movement and a design philosophy characterized by raw materials, stark functionality, and bold geometric forms. This list covers everything brutalism: from concrete buildings to neobrutalist UI frameworks, typography, games, documentaries, and community resources.

## UI Libraries & Frameworks

*CSS frameworks and component libraries for brutalist design*

### CSS Frameworks

- [Brutalist Framework](https://github.com/pinecreativelabs/Brutalist-Framework) - HTML/CSS/JS framework with docs and visual previews
- [NeoBrutalismCSS](https://github.com/matifandy8/NeoBrutalismCSS) - Lightweight (<100KB) minimalist CSS library
- [Tiny Brutalism CSS](https://github.com/pruger/tiny-brutalism-css) - Ultra-minimal framework (<4KB) | [Demo](https://pruger.github.io/tiny-brutalism-css/)
- [Concrete.css](https://concrete.style/) - Classless CSS with automatic dark mode

### Component Libraries

- [Neobrutalism Components (ekmas)](https://github.com/ekmas/neobrutalism-components) - React/Tailwind component collection
- [RetroUI](https://retroui.dev) - Open-source neobrutalist React UI kit with Tailwind
- [neobrutalism-vue](https://github.com/marko-malinic/neobrutalism-vue) - 30+ Vue 3 components with shadcn-vue
- [Neobrutalism shadcn/ui Components](https://shadcn.io) - React UI library with neobrutalism styling

## Architecture & Preservation

*Organizations, databases, and advocacy for brutalist buildings*

### Preservation Organizations

- [SOS Brutalism](https://www.sosbrutalism.org/) - Global database of 2000+ buildings, red list of endangered projects
- [The Twentieth Century Society](https://c20society.org.uk/brutalism) - UK-based conservation authority
- [UMass BRUT](https://www.umassbrut.org) - Award-winning campaign for UMass brutalist architecture
- [Conserving Black Modernism](https://www.getty.edu/foundation/initiatives/current/keeping_it_modern/black_modernism.html) - Getty Foundation initiative preserving Black architects' brutalist work

### Iconic Buildings

- [Barbican Centre](https://www.barbican.org.uk/) - London's brutalist arts complex
- [Habitat 67](https://www.habitat67.com/) - Montreal's modular housing landmark with 2025 guided tours

## Books & Publications

- [Brutalist Plants - Olivia Broome](https://www.waterstones.com/book/brutalist-plants/olivia-broome/9781914314454) - Intersection of plants and concrete
